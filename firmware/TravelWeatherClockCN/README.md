# 旅行天气时钟 - Feather ESP32-S3 TFT 复刻版

面向 Adafruit Feather ESP32-S3 TFT（240x135）的原创兼容实现。

## 功能

- 朝来市实时天气与未来三天预报
- 日本时间 NTP 同步
- 随昼夜、雨、雪切换的像素风旅行场景
- 移动旅行者动画及雨雪粒子
- 动态场景、三天预报、纯数字时钟三个页面
- 短按 BOOT 切换页面
- 长按 BOOT 四秒清除 Wi-Fi
- 首次启动热点：`TravelClock-Setup`

天气数据来自无需 API Key 的 Open-Meteo。像素背景为本移植版本原创素材，
没有复制视频中的角色、游戏或第三方美术资产。

## 与参考项目的关系

参考项目使用 ESP32-C3 和 320x170 屏幕，附件仅提供已编译固件，无法直接
运行在 ESP32-S3 上。本目录为根据公开功能重新实现的兼容版本。
