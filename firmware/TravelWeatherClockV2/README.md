# 旅行天气时钟 V2

针对 Adafruit Feather ESP32-S3 TFT（240x135）重做的横版动漫像素风版本。

## 动画设计

- 固定骑手、场景向后运动的 2D 横版游戏构图
- 24 张统一基线和中心的独立骑行动作帧，动画引擎以 24 FPS 渲染
- 608 像素宽循环全景以约 33 像素/秒滚动，约 18 秒无缝循环一次
- 屏幕保持 24 FPS；24 姿态骑行动画以约 8 姿态/秒播放，完整踩踏周期约 3 秒
- 太阳与月亮由当前时间独立定位，不随地景滚动速度改变
- 雨天使用远、中、近三层雨丝，具有不同长度、亮度、速度、风偏和落地飞溅
- 运行时关闭板载 RGB NeoPixel 与 GPIO13 用户指示灯，保留 TFT 屏幕背光
- 底栏显示 Wi-Fi 信号状态；当天为日本法定祝日时显示“祝日”，否则隐藏
- 温度与湿度在底栏右侧自动靠右对齐
- 时间冒号每秒闪烁；顶部温度范围和底部当前温度均显示 °C
- 右上角同时显示天气文字与像素图标；支持晴天、多云、转晴、小中大雨、雨夹雪、大小雪、雷雨、雾、大风和台风级风雨
- 自然泥土道路，不含车道白线或铺装路面
- 石块、野花与疏密不一的草丛随全景自然循环
- 时间与天气分层组合：白天/晨昏/夜晚可分别搭配云、雨、雪、大风、强风雨或台风级风雨
- 使用朝来市当天日出日落时间计算太阳实时高度，并结合云量、降水和太阳高度调整画面亮度
- 日出与日落加入暖色环境光，中午转为白光；太阳保留分层光晕
- 根据真实日期估算月相和月亮位置；降雨或厚云时隐藏月亮
- 凌晨 00:00 至日出切换为帐篷、篝火和停放自行车的休息场景，同时停止地景滚动
- 根据体感温度、风速和降雨动态切换短袖、衬衫、毛衣、夹克、大衣或雨衣配色与细节
- 两秒场景交叉渐变
- 太阳和月亮沿天空弧线运行，星星动态闪烁
- 动态雨线、飘雪和雷雨闪电
- 天气网络请求在另一个 CPU 核心执行，不阻塞动画

## 操作

- 默认根据朝来市实时天气和日本时间自动切换
- 短按 BOOT：依次预览自动、晴天、夕阳、夜晚、雨天、雪天
- 长按 BOOT 四秒：清除 Wi-Fi 配置
- 首次配网热点：`TravelClock-Setup`

## 美术来源

所有场景、骑手和营地均为本版本原创生成素材。用户提供的截图仅用于构图、动画
需求和像素风格参考，没有复制其中的角色、UI 或第三方游戏素材。
